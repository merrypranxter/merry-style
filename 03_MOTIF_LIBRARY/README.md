## Motif clustering principle

Motifs work best in **thematic clusters** of 2–4 related elements rather than random combinations.

**Crystal temple cluster:** crystal clusters + mirrored pillars + reflective floor + prism light

**Moon garden cluster:** glowing moss + pastel mushrooms + moon disc + reflective pool

**Cosmic initiation cluster:** portal + staircase + aura rings + silver robes

**Glam ecology cluster:** jewel-toned mushrooms + velvet moss + crystal dew + butterflies

See `data/prompt_templates.yaml` for `motif_clusters` section.

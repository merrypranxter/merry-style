# Reference Gallery

This directory will contain curated image analysis, case studies, and moodboard breakdowns for New Age Glam Fantasy.

---

## Purpose

The reference gallery provides concrete visual analysis of specific images, film stills, covers, and artworks that exemplify the style.

Unlike abstract definitions, these entries work from specific visual objects and break them down in terms of:
- Which pillars are active
- How the palette functions
- Which motifs are present
- What the analog texture is doing
- What the emotional tone achieves
- Why it works (or where it drifts)

## How to contribute a reference entry

A reference entry should include:

1. **Image source** — title, artist/creator, year, format
2. **Pillar inventory** — which of the 7 pillars are active and how
3. **Palette breakdown** — specific hues present and their function
4. **Motif list** — symbolic elements present
5. **Analog texture** — which artifact types appear and at what intensity
6. **Emotional tone** — what feeling the image achieves
7. **Style notes** — what makes it on-model or where it drifts

See `../CONTRIBUTING.md` for full contribution guidelines.

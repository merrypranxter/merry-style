# Contributing to New Age Glam Fantasy Style Bible

Thank you for your interest in contributing to this project!

---

## How to Contribute

### Reporting Issues

If you find errors, inconsistencies, or areas for improvement:

1. Check if the issue already exists
2. Provide specific examples
3. Suggest corrections or improvements
4. Reference source material when possible

### Suggesting Additions

Good contributions include:

- **Direct visual analysis** - Break down what makes specific images work
- **Side-by-side comparisons** - Show on-model vs off-model examples
- **Prompt improvements** - Better ways to achieve specific effects
- **Source citations** - Films, books, artists, archives that influenced the style
- **Motif definitions** - New or expanded symbolic elements
- **Technical breakdowns** - How to recreate specific effects
- **Artifact recreation methods** - Post-processing techniques
- **Historical connections** - Links to specific eras or movements
- **Anti-examples** - Images that show style drift

### Content Guidelines

When contributing content:

1. **Be specific** - Avoid vague moodboard language
2. **Be visual** - Describe what things look like, not just how they feel
3. **Be practical** - Include actionable prompts or techniques
4. **Be consistent** - Follow the existing document structure
5. **Be sourced** - Cite influences, references, and examples

### Document Structure

Follow this format for new entries:

```markdown
# Title

## Definition
What this thing is.

## Why it matters
Why it is important to the style.

## Visual characteristics
How it looks in practice.

## Prompt language
Useful phrases and descriptors.

## Common mistakes
How people get it wrong.

## Examples
Sample prompt fragments or scene descriptions.
```

---

## Style of Contribution

### Writing Style

- Use clear, direct language
- Prefer visual description over abstract concepts
- Include specific examples
- Balance theory with practical application

### Tone

- Informative but accessible
- Enthusiastic but analytical
- Respectful of the style's sincerity

## Review Process

Contributions will be reviewed for:

1. Accuracy to the style definition
2. Clarity and usefulness
3. Consistency with existing documentation
4. Proper sourcing and attribution

---

## Code of Conduct

- Be respectful and constructive
- Focus on the work, not the person
- Welcome diverse perspectives
- Maintain the style's sincerity and vision

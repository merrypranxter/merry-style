# Assets

This directory stores visual assets supporting the New Age Glam Fantasy style documentation.

---

## Planned content

- `palettes/` — Color palette swatches and palette diagrams
- `diagrams/` — Visual system diagrams (DNA strands, pillar interaction, boundary maps)
- `reference-stills/` — Curated reference frame captures and analysis images (with proper attribution)
- `banners/` — Repository banners and header images
- `examples/` — Example prompt output images

---

## Asset guidelines

All assets added to this directory should:
1. Be clearly named with format: `[category]-[descriptor].[ext]`
2. Include an attribution note if from external sources
3. Respect copyright — only include properly licensed or original assets
4. Be relevant to illustrating a specific style component

---

## Color palette files

The primary machine-readable color data is in `../data/color_palette.csv` and `../data/style.json`.

Visual swatch files will be added to `palettes/` as the repository grows.

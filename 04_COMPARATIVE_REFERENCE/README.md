# Comparative Reference

This directory documents how New Age Glam Fantasy relates to adjacent, overlapping, and opposing aesthetic traditions.

---

## Purpose

Understanding where a style ends is as important as understanding where it begins.

This section provides structured comparisons, boundary analyses, and anti-reference guides to help maintain style accuracy when working with related aesthetics.

---

## Coverage

Full comparative analysis is documented in:

- **`../01_STYLE_FOUNDATIONS/style-boundaries.md`** — Complete boundary map with 8 detailed comparisons, inside/outside matrix, red/green flag vocabulary, and boundary correction methods.
- **`../data/style_comparison.csv`** — Structured comparison data with relationship type, shared traits, key differences, and drift risk ratings.
